# Agent Rules

- Read `CONTEXT.md` and the applicable `docs/standards/` documents before changing files.
- Keep the inspiration repository read-only and unchanged.
- Keep planner/governance files under `docs/`; keep deployable site code in page directories and `assets/`.
- Use relative, repository-base-path-safe links and assets.
- Do not add a production server, framework runtime, database, authentication, or fabricated personal content.
- Commit only explicitly scoped paths for the current workorder phase.
